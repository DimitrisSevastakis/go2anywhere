Go2Anywhere

Welcome to the go2anywhere wiki!

Go2Anywhere is a chrome extension. It allows you to quickly search through your bookmarks, open tabs or history.

To get quicker access to the plugin add a shortcut from the extensions settings page!

To navigate between the three search sections you can use the Ctrl+ left/righ arrow keys. Additionally using just the up/down keys (without Ctrl) will select the previous or next item in the list.

There are four modes of search*. Default: Searches for exact match on title and exact match on url @: Adding the '@' symbol before the query, it will return only items with exact match on the title, ignoring the url.

: Adding the '#' symbol before the query, it will return only items with exact match on the url, ignoring the title.

: Adding the '>' symbol before the query, it will return only bookmark items that are in a folder path that contains the query string, ignoring url and title.

*The first three mode are applicable to the 'tabs' and 'bookmarks' search while the last one is applicable only on 'bookmarks' search. The history search uses the build in chrome function for searching history.
